## 🚀 Quick Start

### Prerequisites

- Python 3.11+
- Jupyter Notebook (optional)

### Setup

```bash
# Clone the repository
git clone https://github.com/Samar-Khalid/AI-BI-Learning.git
cd AI-BI-Learning

# Install dependencies
pip install -r requirements.txt

# Start Jupyter (optional)
jupyter notebook
```
